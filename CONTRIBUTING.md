# 🤝 Contributing to Awesome Base

Thank you for helping grow this list! Here's how to contribute.

## How to Add a Project

1. **Fork** this repository
2. **Edit** `README.md` and add your project to the right section
3. Use this format:
   ```
   | **Project Name** | Short description (1 sentence) | [Website](https://url.com) |
   ```
4. **Submit a Pull Request** with a brief description

## Guidelines

- ✅ Project must be **live on Base mainnet**
- ✅ Must have a working website
- ✅ Description should be clear and neutral (not marketing fluff)
- ✅ Add to the most relevant category
- ❌ No scam or rug-pull projects
- ❌ No projects under 1 month old (unless exceptional)

## Suggesting a New Category

If you think a whole new category is needed, open an **Issue** first and describe why.

## Reporting Dead Links

Found a broken link? Open an issue with:
- The project name
- The broken URL
- The correct URL (if you know it)
